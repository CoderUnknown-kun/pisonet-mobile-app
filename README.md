# Pisonet Mobile App

Flutter mobile application used to monitor and control pisonet computers remotely.

## Features

- View PC status in real time
- Remote monitoring
- Live screen viewing
- Send commands to PCs

## Tech Stack

- Flutter
- Firebase
- WebRTC

## Related Components

This app communicates with:

- **pisonet-remote-agent** running on each PC
- **pisonet-admin-dashboard** web management panel
